--Different types of HTML Tags--




1. < img >

Copying the image path from any source allows the img to work, Therefore, the img tag is used to embed a specific picture in to a html page.

Example:

< img src = "img_ball.jpg" >




2. < h1 to h6 >

The h1 to h6 represents as the headings in the html, with h1 being at the smallest size and h6 being the biggest.

Example: 

 < h1 > heading 1 < h1 >
          < h2 > heading 2 < h2 >
          < h3 > heading 3 < h3 >
          < h4 > heading 4 < h4 >
          < h5 > heading 5 < h5 >
          < h6 > heading 6 < h6 >



3. < html >

this tag defines the whole file as an html file.

Example:

       < !DOCTYPE html >
           < html lang="en" >
               < head >
                    < title > Title of the document 
                
                < body >

                   < h1 > This is a heading < h1 >
                    < p > This is a paragraph.



4. < !--..-- > Comment Tag

Comments are not displayed in the browser itself, it is only used add a description in order to explain a specific code.

Example: 

        < !--add any type of comment you want-- >

5. < head >

This tag is basically used to define the head section of an HTML file.

Example: 

        < head >
            < title > Title of the document



6. < title >

The title tag allows you to name the web page according to what you want.

Example: 

          < title > Title of the document



7. < p >

10. < div >

The < div > is used to contain different HTML elements which then can be styled in the CSS or JavaScript File.

Example:

            < h2 > a heading in the div 
            < p > a paragraph in the div





--Different CSS Properties--




1. background-color

the background-color allows the assigned tags to have a background color.

Example: 

          body {
            background-color: white;
          }



2. font-size

font-size changes the sizes of the fonts in that specific tags.

Example: 

          body {
            font-size: 10px;
          }



3. font-style

styling the fonts with font-size lets it change all the fonts in that tag.

Example: 

          body {
            font-style: inter;
          }



4. color

adding a simple hex code to most CSS properties will add a color.

Example: 

          body {
           color: coralblue;
          }



5. border-radius

This allows you to have a smoother and rounded corners of an HTML element.

Example: 

          video {
           border-radius: 15px;
          }



6. border

Adds borders to HTML element.

Example: 

          img {
           border: 12px solid #FFFFFF;
          }



7. background-image

It allows you to choose any image url and use that url as the background image.

Example: 

          body {
           background-image: url("...jpg");
          }



8. height and width

This allows you to adjust the height and width of a specific HTML element.

Example: 

          img {
           height: 600px;
           widht: 900px;
          }
